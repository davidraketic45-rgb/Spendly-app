# Spendly - Personal Expense Tracker

A personal expense tracker app with Google Sheets sync.

## Features
- Add, edit, delete expenses
- Google Sheets sync
- Offline support
- Insights & charts
- Date filters
- Payment mode breakdown

## Build
APK is automatically built by GitHub Actions on every push to main branch.
Go to Actions tab → latest run → download Spendly-APK artifact.
